# 校园班级管理分
***
适用于校园班级（整体）周流动分管理网站  
>[!Warning]
>该项目使用deepseek制作，可能会有很多BUG，目前已知显示不正常但是不影响使用。  
***
## 一、部署与使用说明  
1. 环境要求：PHP 7.4+，MySQL 5.7+，启用 PDO。  
2. 导入数据库：将 database.sql 导入 MySQL。  
3. 配置连接：修改 config.php 中的数据库用户名、密码。   
4. 访问系统：将文件部署到 Web 服务器，访问 index.php 登录。
***
## 核心功能覆盖清单  
- [x] 小数点计分  
- [x] 快捷扣分/加分（预设奖惩类型，可临时调分）  
- [x] 学期/周/月管理（自动根据学期开始日期计算周次、月次，可手动切换学期）  
- [x] 按班级排名（周榜、月榜、学期榜，分年级展示）  
- [x] 角色权限（超级管理员、年级管理员、班主任，数据隔离）
>[!Warning]
>班主任~~暂时~~没有任何权限，~~充当访客账号~~    
- [x] 移动端适配（响应式布局，触摸友好）  
- [x] 数据备份与恢复（导出SQL文件，上传恢复）  
- [x] 操作日志（记录管理员关键操作及IP）  
- [x] 密码强度与锁定（前端提示 + 后端验证，5次失败锁1分钟）  
- [x] 班级负责人记录（class_leader 字段）  
- [x] 冻结班级（阻止积分操作，排行榜显示冻结标记）  
- [x] 自定义奖惩类型（管理员可添加）  
- [x] 上传图片  
- [x] 增加/删除班级（可批量使用csv模板导入）  
- [x] 访客查看  
- [x] 支持导出积分记录   
- [x] 二次验证
