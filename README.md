## 一、部署与使用说明  
1.环境要求：PHP 7.4+，MySQL 5.7+，启用 PDO。  
导入数据库：将 database.sql 导入 MySQL。  
配置连接：修改 config.php 中的数据库用户名、密码。  
创建超级管理员：手动在 admins 表插入一条记录，密码使用 password_hash('YourPass123', PASSWORD_DEFAULT) 生成，角色设为 super_admin。（对的，目前这个版本第一次使用要手动生成密码哈希值导入MySQL，看看下一个版本能不能解决这个问题）  
访问系统：将文件部署到 Web 服务器，访问 index.php 登录。

## 核心功能覆盖清单  
✅ 小数点计分（points DECIMAL(5,1)）  
✅ 快捷扣分/加分（预设奖惩类型，可临时调分）  
✅ 学期/周/月管理（自动根据学期开始日期计算周次、月次，可手动切换学期）  
✅ 按班级排名（周榜、月榜、学期榜，分年级展示）  
✅ 角色权限（超级管理员、年级管理员、班主任，数据隔离）  
✅ 移动端适配（响应式布局，触摸友好）  
✅ 数据备份与恢复（导出SQL文件，上传恢复）  
✅ 操作日志（记录管理员关键操作及IP）  
✅ 密码强度与锁定（前端提示 + 后端验证，5次失败锁1分钟）  
✅ 班级负责人记录（class_leader 字段）  
（未知） 冻结班级（阻止积分操作，排行榜显示冻结标记）  
✅ 自定义奖惩类型（管理员可添加）  
✅上传图片  
✅增加/删除班级（可批量使用csv模板导入）  
✅访客查看  
✅支持导出积分记录
